#########Special notes#########
There are two 'master' branches in this repository, the default one is for EDW development; 'ingest_master' is for ingest development.
Please be aware of it, when you need to create branches of from 'master'.
For EDW development, the release branches must be created off from 'master', named as 'DW_CR_XXXX';
For Ingest development, the release branches must be created off from 'ingest_master', named as 'SG_CR_XXXX'.
